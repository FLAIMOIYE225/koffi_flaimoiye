Site web pour un restaurant

Titre : “Pizza Mama”

Le projet “Pizza Mama” est une application web conçue pour une pizzeria virtuelle. Grâce à une combinaison
de technologies telles que HTML, CSS et Python, j'ai eu a créer un menu interactif et visuellement attrayant
pour les amateurs de pizza.

Essayez dès maintenant “Pizza Mama” et laissez-vous emporter dans un menu varier de pizzas !

Technologies utilisées

HTML CSS DJANGO PYTHON PythonAnyWhere
